# IF Library

원작 기반 이야기를 선택과 장면 연출로 체험하는 비주얼 노벨 라이브러리입니다.

## 공개 플레이

- 옹고집전 바로 읽기: https://luckybridge.github.io/if-library/?story=review-main-003&play=1
- 스토리 목록: https://luckybridge.github.io/if-library/

## 로컬 실행

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 5173
```

## 검증

```bash
npm run validate:stories
npm run lint
npm run build
```

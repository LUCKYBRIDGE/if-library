# review-main-002 이야기 검토용 읽기본 (개발자/작가 연출본)

이 파일은 `review-main-002`를 비주얼 노벨(미연시) 개발 및 연출 관점에서 쉽게 빌드하고 검토할 수 있도록 **배경(BG), 배경음악(BGM), 효과음(SFX), 인물 스프라이트 표정**을 구조화하여 작성한 최종 대본이다.
원본 기준 파일은 `src/data/adventureApprovedStories.js`이며, 이 이야기는 아직 승인본이 아니라 `review-candidate` 상태다.

## 기본 정보
- ID: review-main-002
- 제목: 무거운 신뢰의 저울
- 상태: review-candidate
- 카테고리: folktale-adaptation
- 장소/시간 시작 라벨: 용궁 대청 / 저녁, 육지로 떠나기 전
- 원전/모티브: 토끼전/별주부전/수궁가

## 연출 기획 및 작가 의도
1. **사운드 디자인 (BGM/SFX)**: 
   - 용궁의 깊은 바닷소리와 황량한 육지의 찬 바람 소리를 SFX로 대비시켜 청각적 고립감을 줍니다.
   - BGM은 정적이고 장엄한 동양풍 현악기 중심에서, 위기 시 가빠지는 리듬으로, 결말에서는 쓸쓸한 피아노와 바이올린 독주로 전환하여 감정선을 이끕니다.
2. **비주얼 연출 (스프라이트 표정)**:
   - 자라(주인공)의 눈빛 변화(충직 → 번민 → 침통), 용왕의 안색 변화(파리함 → 고통 → 미소), 토끼의 표정 변화(경계 → 사색 → 절박 → 원망 → 안도)를 지문으로 묘사하여 화면 스크립트 작성에 바로 반영할 수 있도록 합니다.
3. **텍스트 딜레마 빌드업**:
   - 모호하거나 문학적인 은유는 지우되, 상황의 현실적인 무게(왕의 통치 책임 vs 무고한 짐승의 생명권)를 인물의 직관적인 대화에 담아 플레이어가 깊은 고민 끝에 선택지를 누르도록 설계했습니다.

# IF Library 제작 지침

이 폴더에는 다음 인터랙티브 소설을 만들 때 직접 참고할 최신 지침만 둔다.

## 지침 문서

- [미연시 화면/연출 제작 지침](./visual-novel-ui-guidelines.md)
  - 이미지, 배경, 캐릭터 스프라이트, 글상자, 버튼, 선택지, 회상, 전환 효과 기준
- [미연시 대본/줄거리 제작 지침](./visual-novel-script-guidelines.md)
  - 등장인물, 말투, 사건 흐름, 컷 분리, 선택지, 결말, 생각해보기 질문 기준

## 문서 관리 기준

- `docs/`에는 최신 제작 지침과 의사결정 기준만 남긴다.
- 원본 대본이나 가져온 소스 데이터는 `src/data/source/`를 기준으로 확인한다.
- 실제 플레이 화면과 표시 문구는 `src/data/stories.js`, `src/components/`, `src/styles.css`를 기준으로 확인한다.
- 과거 화면 캡처, 임시 검증 이미지, 오래된 원본 복사본은 지침으로 남기지 않는다. 현재 UI와 어긋나면 다음 작업자를 혼란스럽게 만들 수 있기 때문이다.
